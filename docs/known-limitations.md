# Limitaciones conocidas

- La sincronizacion entre dispositivos depende de Firebase y de que las reglas esten publicadas.
- La PWA no garantiza ejecucion en segundo plano si el navegador esta cerrado.
- La exportacion estatica requiere ajustar `basePath` para GitHub Pages en subdirectorio.
- Los graficos se presentan como tarjetas y tablas accesibles en esta primera version.
